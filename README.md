# Ubuntu Workstation Setup

Documentation for setting up a new Ubuntu development workstation.

This repository records the steps, tools, and configuration used when preparing a fresh machine.  
The goal is reproducibility: a new computer should be configurable by following these steps.

---

# Setup Order

Follow the steps in this order.

1. System update  
   docs/01-system-update.md

2. Package systems (APT / Snap)  
   docs/02-package-systems.md

3. Core tools  
   docs/03-core-tools.md

4. Git setup  
   docs/04-git-setup.md

5. Desktop applications  
   docs/05-desktop-apps.md

6. Development tools  
   docs/06-dev-tools.md

7. System configuration  
   docs/07-system-config.md

# Philosophy

Rules for maintaining this repository:

• Document unusual install steps with comments.  
• Prefer official repositories when possible.
